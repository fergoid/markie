git local & remote


1. New git repo  
    git init    
3. Add files to local repo  
    git add . --all  
5. Check status  
    git status  
7. First commit  
    git commit -m 'Initial commit'  
9. Add remote
    git remote add origin https://github.com/fergoid/markie.git  
11. Push from local to remote
    git push origin master
